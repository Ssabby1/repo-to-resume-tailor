# Codex Installation

## Install

Copy the inner `repo-to-resume-tailor/` folder into your Codex skills directory:

```text
~/.codex/skills/repo-to-resume-tailor/
```

The installed layout should contain:

```text
SKILL.md
agents/openai.yaml
references/prompt.md
references/examples.md
```

## Usage

Example:

```text
Use $repo-to-resume-tailor to analyze this repository and write one resume-ready project description tailored to an AI engineer role, with a dedicated tech stack line.
```

You can also provide a concrete job description and ask for a JD-tailored version.
